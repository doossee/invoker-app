# ⚡ Invoker App

**A Bruno-like API client for the web.**

Built with React, CodeMirror 6, and [ivkjs](https://github.com/doossee/ivkjs).

</div>

---

## Development

```bash
git clone https://github.com/doossee/invoker-app.git
cd invoker-app
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```
